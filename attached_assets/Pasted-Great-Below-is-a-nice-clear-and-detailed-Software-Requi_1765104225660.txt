# 1. **Introduction**

## 1.1 **Project Overview**

This is a web-based e-commerce platform for selling handmade crochet products such as bags, sweaters, tops, dolls, hats, and accessories. Customers will browse products, view details, add items to a shopping cart, make orders, and track deliveries in real-time‚Äîsimilar to Kilimall.

The website will have both a customer-facing interface and an admin dashboard for managing products, orders, and deliveries.

## 1.2 **Purpose**

The purpose of this system is to enable online selling of locally handmade crochet items, provide an easy shopping experience, and allow customers to track orders from purchase to delivery.

## 1.3 **Users**

* **Customers** (buyers)
* **Admin** (store owner)
* **Delivery team** (optional ‚Äî updates delivery status)

# 2. **System Features & Requirements**

---

# 2.1 **Customer-Facing Features**

## **2.1.1 Homepage**

* Modern, clean design using yellow & green theme.
* Carousel or banner showing featured crochet items.
* Quick links to categories (bags, clothes, dolls, accessories).
* Display newly arrived products.
* Display top-selling products.

## **2.1.2 Product Catalog**

* Shows product list with:

  * Product image
  * Name
  * Price
  * "Add to cart" button
* Filter products by:

  * Category
  * Price range
  * Availability
* Search bar (search by name/keyword).

## **2.1.3 Product Details Page**

* Product images (multiple if needed)
* Name, price, stock status
* Product description and specifications
* Available colors/sizes (if any)
* Reviews/ratings (optional)
* Add to cart button

## **2.1.4 Shopping Cart**

* View all added items
* Increase/decrease quantity
* Remove items
* Display subtotal and total
* Estimate delivery timeline

## **2.1.5 Checkout**

* Customer details:

  * Full name
  * Phone number
  * Delivery address
* Payment methods:

  * Cash on Delivery
  * MPESA (optional integration)
* Order summary
* Place order button

## **2.1.6 Order Confirmation**

* After placing an order:

  * System generates an **Order Code** (e.g. CR-2025-001)
  * Customer sees success message
  * Customer can track their order using the order code

## **2.1.7 Order Tracking Page**

Customers can track order status:

1. **Pending**
2. **Processing**
3. **Packed**
4. **On the Way**
5. **Delivered**

Each status shows:

* Date & time updated
* Visual progress bar in yellow and green

## **2.1.8 Customer Account (optional)**

* Register/login
* Update profile
* View past orders
* Track orders faster
